About the site

Under the hood, everything is neatly tied together with Kohana. Content is written in GitHub Flavored Markdown and presented using AnimationFrame, canvgc, Grunt, Modernizr, Normalize.css, Picturefill, Respond.js, ScrollUp, Selectivizr, svg2png and svgstore. Screen capture videos were created with VirtualDub, UScreenCapture and Ut video codec.

Heading typeface is Titillium by Campi Visivi.
